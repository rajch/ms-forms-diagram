Forms Diagram v1.1.0

Bienvenue

Cette extension peut vous aider à gérer la logique de branchement complexe dans Microsoft Forms, en affichant vos branchements dans un diagramme de flux facile à suivre.

Pour commencer

  1. Pour de meilleurs résultats, épinglez/affichez l'extension sur/dans la barre d'outils de votre navigateur. Au départ, elle apparaîtra grisée, comme ceci :
  2. Ensuite, ouvrez un formulaire à partir du site Web Microsoft Forms.
  3. Mettez l'éditeur de formulaire en mode "Options de branchement" en choisissant "Ajouter un embranchment" dans le menu contextuel de n'importe quelle question. L'extension s'allumera, comme ceci :
  4. Cliquez sur l'icône de l'extension pour voir votre logique de branchement sous forme d'un organigramme.

Nouvelles fonctionnalités dans la v1.1.0

  • Le diagramme de flux utilise désormais la couleur de votre thème actif.
  • Copiez le diagramme dans le presse-papier en un clic.
  • Copiez le code source du diagramme dans le presse-papier, pour le modifier/l'intégrer partout où les diagrammes mermaid sont autorisés.
  • Choisissez parmi quatre styles de dessin de ligne d'organigramme différents.
