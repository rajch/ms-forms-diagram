Forms Diagram v1.2.0

Welcome

This extension can help you manage complex branching logic in Microsoft Forms, by showing your branches in an easy-to-follow flowchart diagram.

Getting Started

  1. For best results, pin/show the extension to/on your browser toolbar. Initially, it will appear grayed out, like so:
  2. Next, open a form from the Microsoft Forms web site.
  3. Put the form editor in "Branching Options" mode, by choosing "Add branching" from the context menu of any question. The extension will light up, like this:
  4. Click the extension icon to see your branching logic as a flowchart.

Features

  • Choose from three different color schemes, including your active theme color.
  • Choose from four different flowchart line drawing styles.
  • Copy the diagram to the clipboard in one click.
  • Copy the source code for the diagram to the clipboard, for editing/embedding anywhere where mermaid diagrams are allowed.
  • Download the diagram as a PNG file.
